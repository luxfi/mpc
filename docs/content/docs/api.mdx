# API Reference

```go
import "github.com/luxfi/mpc/pkg/threshold"
```

```go
type UnifiedThresholdAPI struct {
    // ...
```

### Scheme

```go
type Scheme string

const (
    SchemeECDSA   Scheme = "ecdsa"
    SchemeEdDSA   Scheme = "eddsa"
    SchemeTaproot Scheme = "taproot"
)
```

### Party

```go
type Party interface {
    ID() string
    PublicKey() []byte
    Sign(message []byte) ([]byte, error)
}
```
